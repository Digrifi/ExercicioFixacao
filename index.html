  • Python é uma linguagem de código aberto de forma fácil e acessível de aprender. Ela também é eficiente para projetos complexos, pois sua legibilidade e sua sintaxe são simples e diretas. Entendendo-a é possível programar para computadores, celulares e jogos, entre aplicações básicas e avançadas. Sua capacidade de processar grandes bases de dados acaba a levando para o meio da computação científica, ciência de dados e Inteligência Artificial.
